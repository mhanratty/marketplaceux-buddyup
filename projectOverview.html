What is BuddyUp?

BuddyUp is a prototype concept for a live help service that provides instant, personalized support to Firefox OS users via chat, screen sharing and remotely controlling the user’s device.

Why this is important

  • Differentiates from Android support. Instead of explaining a solution, show me how to solve my problem through screen sharing.
  • Builds on Mozilla’s community-driven support model to create a large-scale peer-to-peer service that reduces support costs for carriers.

How it Works

  • BuddyUp connects a Firefox OS user with a Helper.
  • User can grant Helper permission to view their screen and remotely control their device
  • Helper shows user how to resolve the problem the user is having on device

Watch the demo: https://www.youtube.com/watch?v=V4--f8G8DAs&feature=youtu.be

Who are the Helpers?

  • You can get help from someone in your contacts, a carrier support agent, or a Firefox OS community member
  • The Helper can be on any device that has a browser including desktops on Android devices.
  • Helpers only have certain privileges on the device. For example, they cannot install apps on the device.
