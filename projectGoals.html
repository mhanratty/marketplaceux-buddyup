Goals

  • Differentiate – Make it quick and easy to get help from your buddy in a chat with screen sharing. Instead of explaining a problem, just 'show' someone remotely and get instant, friendly support.
  • Create User Value – Reduce the anxiety associated with moving to a smartphone with a new operating system.
  • Scale to Support Millions - Build on Mozilla’s community-driven support model to create a large-scale peer-to-peer service that reduces support costs for carriers.

Success Metrics

  • Increase number of people getting 1:1 help for a Firefox OS issue from current state of asking question in forum
  • Increase user satisfaction from 50% with articles (exit survey from support.mozilla.org) to 80-90% with live help
  • Increase user satisfaction from 30% from forum posts to 80-90% with live help
  • Reduce number of support calls to operators regarding software issues.
  • Increase user satisfaction for users calling operator support to resolve Firefox OS software issues.
  • Reduce number of minutes the operator call agent is on a support call for a Firefox OS software issue.
